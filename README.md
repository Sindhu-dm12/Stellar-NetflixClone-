# Stellar-NetflixClone-
Stellar — Netflix-Inspired Streaming UI

Stellar is a Netflix-style streaming platform UI clone that showcases a modern OTT interface with dynamic movie rows, a featured hero banner, and responsive design. This project focuses on building an engaging frontend experience using vanilla JavaScript and modern CSS techniques.

Features:
Netflix-inspired modern UI
Dynamic hero banner with featured content
Categorized movie rows (Trending, Top Rated, Action, Comedy, Horror)
Horizontal scrolling movie carousels
Navbar with scroll effect
Responsive layout for mobile and desktop
Dynamic image loading with fallback handling
Clean and reusable component structure

Tech Stack:
HTML5 – Structure
CSS3 – Styling and layout
JavaScript (ES6) – Dynamic rendering and interactions
Font Awesome – Icons
Google Fonts – Typography
